## 8. Label design brief (for a separate Canva session)

This brief is written so it can be handed off to a Canva-focused working session without extra context.

### Purpose

A sticker-size label applied to bottles (beer, wine, spirits) at Folklore bar-service events. Each label personalizes a bottle for a donor at the $25 Dedication tier.

### Label variants (one template, three fill states)

1. **Name only** — when a donor leaves dedicated-to and message blank
2. **Name + dedicated-to** — when a donor fills in the dedicated-to line only
3. **Name + dedicated-to + message** — the fullest version

The template should scale gracefully between these states. A name-only label shouldn't look empty. A fullest-version label shouldn't feel crowded.

### Dimensions

Two sizes, same design language:

- **Wine/spirits label:** approximately 3" wide × 4" tall
- **Beer label:** approximately 2" wide × 3" tall

### Fixed elements (on every label)

- Small Folklore wordmark in one corner, set in Take Me To Tuscany
- The word "DEDICATION" as a subhead or small header mark (Jost, uppercase, letter-spaced)

### Dynamic fields

- **Donor name** — set in Take Me To Tuscany, 18–24pt depending on label size and name length. Needs to handle short names ("Jen") and longer ones ("Christopher Alexander").
- **"Dedicated to" line** — set in Cormorant Garamond italic, smaller than the donor name
- **Short message** — set in Cormorant Garamond italic, smallest text on the label. Needs to handle up to 100 characters gracefully, wrapping across two or three lines as needed.

### Color palette

Pull from the Folklore primary palette. Deep plum (`#2a1f3d`) and copper (`#c17b4a`) as the main two, on a cream or warm-cream ground. Not jokey, not sticker-bomb-y. Labels should read as tasteful, literary, slightly whimsigoth.

### Print spec

- Vinyl or waterproof paper
- Adhesive-backed
- Suitable for application to refrigerated wet bottles
- 300 DPI for print
- Bleed: 1/8" if going to a print shop; none if cutting in-house

### Deliverables from the Canva session

- PNG and PDF exports for each label size (wine and beer)
- A working Canva template that can be duplicated and filled per donor
- A reference sheet showing all three fill states for both sizes

## 9. Operational workflow

Who does what, when. One page.

### Stage 1 — Donation received (within 24 hours)

- Zeffy sends notification to `hello@folkloresocietywa.org`
- Candace (or designated admin) adds the entry to the Dedication Tracker: donor name, email, amount, label text, event assignment, status
- Sends the thank-you email using the template above
- Tracker status: `Queued`

### Stage 2 — Queue closes (~14 days before the event)

- No new dedications accepted for this event after the queue closes
- Dedications received after this point roll automatically to the next scheduled event
- Candace exports the final list for this event's label production
- Tracker status: `Printing`

### Stage 3 — Label production (~10 days before event)

- Candace builds each label in the Canva Dedication template
- Labels printed on vinyl/waterproof label stock
- QA pass: every label read back against the original Zeffy entry before it's cleared for use
- Tracker status: `Printed`

### Stage 4 — Pre-event email (~7 days before event)

- Candace sends the pre-event notification email to each donor, with a photo of their labeled bottle attached
- Tracker status: `Emailed`

### Stage 5 — Event day

- Bottles labeled day-of or day-before (whichever fits the event setup rhythm)
- Dedicated bottles placed in bar rotation alongside regular inventory
- Event photographer (or Candace) captures each labeled bottle in setting
- Empty bottles placed in a designated bin for pickup after service
- Tracker status: `Served`

### Stage 6 — Post-event (within 48 hours)

- Post-event social post with bottle photos
- Individual follow-up email to each donor with their bottle photo and pickup offer
- Tracker status: `Photo sent`

### Stage 7 — Pickup window (30 days post-event)

- Candace coordinates individual pickup appointments with interested donors
- At the 30-day mark, remaining empty bottles are recycled or repurposed
- Tracker status: `Picked up` or `Recycled`

### Tools

- **Dedication Tracker:** Google Sheet for now; migrate to Airtable or similar if volume warrants
- **Design:** Canva (template built in the label design session)
- **Print:** in-house label printer or local print shop; TBD based on volume
- **Email:** Gmail at `hello@folkloresocietywa.org`
- **Photography:** iPhone-quality shots are fine; consistency of framing matters more than gear
